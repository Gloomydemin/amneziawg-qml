# Wireguard

Это форк от https://github.com/DavidVentura/Wireguard_qml

AmneziaWG для Ubuntu Touch

AmneziaWG VPN клиент для Ubuntu Touch. Форк WireGuard_qml с поддержкой AmneziaWG протокола.

## Особенности
- Kernel + Userspace WireGuard
- AmneziaWG протокол (DPI устойчивость)
- Lomiri UI
- Python3 + Go бэкенд

## Сборка
```bash
clickable build --arch arm64
clickable install
```
